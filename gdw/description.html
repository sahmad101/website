HTML Element

HTML Tag
HTML tags begins with the less-than (<) character and end with greater-than (>).
angle bracket
Angle brackets are < and >.
HTML attribute
HTML attributes provide additional information about HTML Elements, they are special words used inside the opening tag to control the element's behavior.
HTML comment
They are text or code written in the code to give an explanation about the code and not visible when opened in the browser.
