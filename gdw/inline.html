Inline Elements

Inline Elements are used to markup text inside a block element like a paragraph or division. Each element does different things to the text. Below are the examples.

Text is a strong element
Text is bolded
Text is emphasized
Text is italicized
Text is in quotation marks
Text is enclosed (source code)
Text is enclosed by the user in keyword
Text is in superscript
Text is in subscript
Text is in inline logical division. No default attributes.
